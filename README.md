# Mangayomi Extensions

This repository contains the available extension catalogues for the Mangayomi app.

## How to add the extensions

Click on one of the buttons below to add the corresponding repository/repositories:

<a href="https://intradeus.github.io/http-protocol-redirector?r=mangayomi://add-repo?repo_name=myomi-xtensions&repo_url=https://github.com/SyedArbaazHussain/myomi-xtensions&manga_url=https://raw.githubusercontent.com/SyedArbaazHussain/myomi-xtensions/main/manga_index.json&anime_url=https://raw.githubusercontent.com/SyedArbaazHussain/myomi-xtensions/main/anime_index.json&novel_url=https://raw.githubusercontent.com/SyedArbaazHussain/myomi-xtensions/main/novel_index.json"><img alt="Add all repositories" src="images/add-all-repositories.png" height="32"></a>

<a href="https://intradeus.github.io/http-protocol-redirector?r=mangayomi://add-repo?repo_name=myomi-xtensions&repo_url=https://github.com/SyedArbaazHussain/myomi-xtensions&manga_url=https://raw.githubusercontent.com/SyedArbaazHussain/myomi-xtensions/main/manga_index.json"><img alt="Add manga repository" src="images/add-manga-repository.png" height="32"></a>

<a href="https://intradeus.github.io/http-protocol-redirector?r=mangayomi://add-repo?repo_name=myomi-xtensions&repo_url=https://github.com/SyedArbaazHussain/myomi-xtensions&anime_url=https://raw.githubusercontent.com/SyedArbaazHussain/myomi-xtensions/main/anime_index.json"><img alt="Add anime repository" src="images/add-anime-repository.png" height="32"></a>

<a href="https://intradeus.github.io/http-protocol-redirector?r=mangayomi://add-repo?repo_name=myomi-xtensions&repo_url=https://github.com/SyedArbaazHussain/myomi-xtensions&novel_url=https://raw.githubusercontent.com/SyedArbaazHussain/myomi-xtensions/main/novel_index.json"><img alt="Add novel repository" src="images/add-novel-repository.png" height="32"></a>


Or add them manually in the app (More -> Settings -> Browse):

manga repo
```
https://github.com/SyedArbaazHussain/myomi-xtensions/blob/main/manga_index.json
```

anime repo 
```
https://github.com/SyedArbaazHussain/myomi-xtensions/blob/main/anime_index.json
```

novel repo 
```
https://github.com/SyedArbaazHussain/myomi-xtensions/blob/main/novel_index.json
```
